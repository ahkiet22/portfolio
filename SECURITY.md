# Security Policy

## Supported Versions

Only the following versions of this project are currently supported with security updates:

| Version | Supported |
| ------- | --------- |
| Latest  | ✔️        |
| Older versions | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability, please follow these steps:

1. Do NOT create a public GitHub issue.
2. Contact me directly at: **[lekiett2201@gmail.com]**
3. Provide as much detail as possible:
   - Steps to reproduce
   - Impact
   - Possible fixes (if any)

I will review the report as soon as possible and work on a fix.  
Responsible disclosures are greatly appreciated.

## Security Best Practices

While using or contributing to this project, please:

- Do not commit secrets, tokens, or passwords.
- Avoid exposing private API keys in code or logs.
- Use environment variables for sensitive configuration.
- Follow safe coding guidelines (validation, sanitization, etc.).

Thank you for helping keep this project secure!
